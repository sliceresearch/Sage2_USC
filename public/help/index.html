Sharing content

You can upload and display content by drag-and-drop onto the zone marked 'Drop multimedia files here': in the SAGE Pointer, in the SAGE UI file zone (green dashed boxe) or the whole wall representation (SAGE UI). Files are uploaded (see the progress bar during upload) and the pertinent application is started.

Supported file types:

  • Images
    • JPEG
    • PNG
    • TIFF
    • BMP
    • PSD
  • Videos
    • MP4
    • M4V
    • WEBM
  • Documents
    • PDF

Voice Commands

When using Google Chrome it is possible to issue voice commands through the UI.
Click and hold the mouse button in the overview area of the UI page. After about one second, an ear will show. This indicates that the speech recognition component is active and should be listening.
If you are doing so for the first time, the browser will ask you for permission to access your microphone. Allowing access to the microphone is necessary to issue voice commands.


To disable voice recognition, the UI menu Services has a Disable Voice Recognition option.
To keep the voice recognition whenever using the wall, block microphone access to the page.


There are two main types of commands: wall and application commands. Commands are checked if they are for the wall first. They do not relate to a specific already open application. Application commands are for a specific application that is already open. Applications can be issued a command corresponding to any entry within its right click menu. Click and hold the mouse button over the application that you want to issue commands.
