Desktop sharing

The first time before using SAGE2, one must enable screen capture support in Chrome by installing a Chrome extension:

  • Navigate to the Chrome Webstore for extensions and install the SAGE2 Desktop Sharing extension (click on the + sign):
    SAGE2 Screen Capture

  • Make sure the extension SAGE2 Screen Capture is installed, enabled and up to date

    Open tab and type the following URL in the address bar: chrome://extensions/

Sharing content

You can upload and display content by drag-and-drop onto the zone marked 'Drop multimedia files here': in the SAGE Pointer, in the SAGE UI file zone (green dashed boxe) or the whole wall representation (SAGE UI). Files are uploaded (see the progress bar during upload) and the pertinent application is started.

Supported file types:

  • Images
    • JPEG
    • PNG
    • TIFF
    • BMP
    • PSD
  • Videos
    • MP4
    • M4V
    • WEBM
  • Documents
    • PDF
